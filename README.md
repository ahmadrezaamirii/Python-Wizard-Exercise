### Define the sets
````
SetA = {1, 2, 3, 4}

SetB = {3, 4, 5, 6}
````
### Find union and intersection
````
u = SetA.union(SetB)       # or: u = SetA | SetB
i = SetA.intersection(SetB) # or: i = SetA & SetB
````
### Display the results
````
print(u)
print(i)
````
